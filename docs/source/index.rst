Project Overview
----------------

This software enables extracting and analysis of scientific papers using Grobid and Python

Features
--------

- Abstract extraction
- Keyword cloud generation
- Figure counting
- Link extraction
- Docker support
- Automated tests

For more information and complete instructions, see the README file in the repository.
